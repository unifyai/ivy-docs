Quaternion#

Collection of Rotation Conversion Functions to Quaternion Format

ivy_mech.orientation.quaternion.axis_angle_to_quaternion(axis_angle)#

Convert rotation axis unit vector \(\mathbf{e} = [e_x, e_y, e_z]\) and rotation angle \(θ\) to quaternion \(\mathbf{q} = [q_i, q_j, q_k, q_r]\).n [reference] # noqa

Parameters:

axis_angle – Axis to rotate about and angle to rotate [batch_shape,4]

Returns:

ret – Quaternion [batch_shape,4]

ivy_mech.orientation.quaternion.euler_to_quaternion(euler_angles, convention='zyx', batch_shape=None)#

Convert \(zyx\) Euler angles \(\mathbf{θ}_{abc} = [ϕ_a, ϕ_b, ϕ_c]\) to quaternion \(\mathbf{q} = [q_i, q_j, q_k, q_r]\).n [reference]

Parameters:

  • euler_angles – Euler angles, in zyx rotation order form [batch_shape,3]

  • convention – The axes for euler rotation, in order of L.H.S. matrix multiplication. (Default value = ‘zyx’)

  • batch_shape – Shape of batch. Inferred from inputs if None. (Default value = None)

Returns:

ret – Quaternion [batch_shape,4]

ivy_mech.orientation.quaternion.get_random_quaternion(max_rot_ang=3.141592653589793, batch_shape=None)#

Generate random quaternion \(\mathbf{q} = [q_i, q_j, q_k, q_r]\), adhering to maximum absolute rotation angle.n [reference]

Parameters:

  • max_rot_ang – Absolute value of maximum rotation angle for quaternion. Default value of \(π\).

  • batch_shape – Shape of batch. Shape of [1] is assumed if None. (Default value = None)

Returns:

ret – Random quaternion [batch_shape,4]

ivy_mech.orientation.quaternion.hamilton_product(quaternion1, quaternion2)#

Compute hamilton product \(\mathbf{h}_p = \mathbf{q}_1 × \mathbf{q}_2\) between \(\mathbf{q}_1 = [q_{1i}, q_{1j}, q_{1k}, q_{1r}]\) and \(\mathbf{q}_2 = [q_{2i}, q_{2j}, q_{2k}, q_{2r}]\).n [reference]

Parameters:

  • quaternion1 – Quaternion 1 [batch_shape,4]

  • quaternion2 – Quaternion 2 [batch_shape,4]

Returns:

ret – New quaternion after product [batch_shape,4]

ivy_mech.orientation.quaternion.inverse_quaternion(quaternion)#

Compute inverse quaternion :math:`\mathbf{q}^{-1}.\n `[reference] <https://github.com/KieranWynn/pyquaternion/blob /446c31cba66b708e8480871e70b06415c3cb3b0f/pyquaternion/quaternion.py#L473>`_

Parameters:

quaternion – Quaternion [batch_shape,4]

Returns:

ret – Inverse quaternion [batch_shape,4]

ivy_mech.orientation.quaternion.polar_axis_angle_to_quaternion(polar_axis_angle)#

Convert polar axis-angle representation, which constitutes the elevation and azimuth angles of the axis as well as the rotation angle \(\mathbf{θ}_{paa} = [ϕ_e, ϕ_a, θ]\), to quaternion form \(\mathbf{q} = [q_i, q_j, q_k, q_r]\).n [reference]

Parameters:

polar_axis_angle – Polar axis angle representation [batch_shape,3]

Returns:

ret – Quaternion [batch_shape,4]

ivy_mech.orientation.quaternion.rot_mat_to_quaternion(rot_mat)#

Convert rotation matrix \(\mathbf{R}\in\mathbb{R}^{3×3}\) to quaternion \(\mathbf{q} = [q_i, q_j, q_k, q_r]\).n [reference] # noqa

Parameters:

rot_mat – Rotation matrix [batch_shape,3,3]

Returns:

ret – Quaternion [batch_shape,4]

ivy_mech.orientation.quaternion.rotation_vector_to_quaternion(rot_vector)#

Convert rotation vector \(\mathbf{θ}_{rv} = [θe_x, θe_y, θe_z]\) to quaternion \(\mathbf{q} = [q_i, q_j, q_k, q_r]\).n [reference] # noqa

Parameters:

rot_vector – Rotation vector [batch_shape,3]

Returns:

ret – Quaternion [batch_shape,4]

ivy_mech.orientation.quaternion.scale_quaternion_rotation_angle(quaternion, scale)#

Scale the rotation angle \(θ\) of a given quaternion \(\mathbf{q} = [ q_i, q_j, q_k, q_r]\).n [reference]

Parameters:

  • quaternion – Quaternion [batch_shape,4]

  • scale – Value to scale by the rotation angle by. Can be negative to change rotation direction.

Returns:

ret – Quaternion with rotation angle scaled [batch_shape,4]
